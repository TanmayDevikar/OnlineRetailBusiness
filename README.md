A full-stack e-commerce application built using Spring Boot and React, designed to simulate a production-grade online shopping system with secure payments, scalable APIs, and performance-optimized frontend delivery.

## Tech Stack

Backend:
- Java
- Spring Boot
- REST APIs
- JWT Authentication
- MySQL

Frontend:
- React
- Redux
- Material UI

Testing & Performance:
- Apache JMeter

Payments:
- Razorpay Payment Gateway

---

## Key Features

### Secure Authentication
JWT-based authentication and role-based authorization for users and administrators.

### Product & Order Management
Complete product catalog, cart management, and order lifecycle handling.

### Payment Integration
Integrated Razorpay payment gateway for secure transaction processing.

### Performance Optimization
Frontend code-splitting and lazy loading reduced initial page load time by ~35%.

### Load Testing
Simulated 10,000+ monthly transactions using Apache JMeter to validate API reliability and stability under load.

### Admin Dashboard
Administrative interface for managing users, products, and orders.
